# MonitCash 💸

Веб-застосунок для обліку особистих фінансів.

## Функціонал

- 🔐 Авторизація (email / телефон + пароль)
- 🏠 Головна — баланс, доходи/витрати, графік, останні транзакції
- 💳 Транзакції — додавання, редагування, видалення, фільтри
- 📊 Аналітика — лінійний, стовпчастий та круговий графіки
- 🎯 Бюджети — ліміти по категоріях з прогрес-барами
- 🏆 Цілі — накопичення на мрії з поповненням
- 👤 Профіль — налаштування, зміна пароля, експорт/імпорт
- 🌓 Темна та світла тема

## Технології

- HTML5 / CSS3 / Vanilla JavaScript
- Chart.js 4.4
- Google Fonts (Sora, JetBrains Mono)
- localStorage для зберігання даних

## Запуск

Просто відкрийте `index.html` у браузері — жодних залежностей встановлювати не потрібно.

## GitHub Pages

Після завантаження на GitHub увімкніть **GitHub Pages** (Settings → Pages → Branch: main) і застосунок буде доступний онлайн.
